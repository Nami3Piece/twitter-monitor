## 阿里云购买指南

### 推荐方案：轻量应用服务器

**访问**: https://www.aliyun.com/product/swas

### 配置选择

1. **地域**：
   - 推荐：香港 或 新加坡
   - 原因：国际访问快，不需要备案

2. **套餐**：
   - **推荐**: 1核2GB 3Mbps
   - 价格：约 ¥24/月 或 ¥288/年
   - 足够运行你的服务

3. **镜像**：
   - 选择：Ubuntu 22.04 LTS
   - 或：Ubuntu 20.04 LTS

4. **时长**：
   - 建议：先买 1 个月测试
   - 确认稳定后再续费年付（更便宜）

### 购买步骤

1. 访问 https://www.aliyun.com/product/swas
2. 点击"立即购买"
3. 选择配置：
   - 地域：香港
   - 套餐：1核2GB
   - 镜像：Ubuntu 22.04
   - 时长：1个月
4. 勾选协议，点击"立即购买"
5. 支付

### 购买后立即操作

1. 登录阿里云控制台: https://swas.console.aliyun.com/
2. 找到你的服务器实例
3. 点击"远程连接" 或 记录服务器的公网IP
4. 设置 root 密码（如果还没设置）

### 获取服务器信息

购买完成后，告诉我：
- 服务器公网 IP: xxx.xxx.xxx.xxx
- Root 密码（或者我会指导你设置）

然后我会：
1. 生成完整的部署命令
2. 你只需要复制粘贴到服务器
3. 5分钟内完成部署
